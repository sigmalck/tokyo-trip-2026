地铁/铁路速记

机场

羽田 ⇄ 五反田

羽田机场 → 品川 → 五反田，行李日优先走少换乘路线。

  1. HND机场
  2. 品川京急
  3. 五反田山手线
约 35-45 分钟；不写死票价，刷 Suica/PASMO 最省心。
Disney

五反田 ⇄ 舞浜

五反田 → 东京 → 舞浜，Disney 日越早越舒服。

  1. 五反田山手线
  2. 东京换京叶线
  3. 舞浜步行入园
约 45-60 分钟；东京站京叶线换乘走得久，别压秒。
东线

浅草 → 晴空塔 → 秋叶原

都营浅草线把五反田和浅草/押上串起来，再折回秋叶原。

  1. 五反田浅草线
  2. 浅草寺社/街区
  3. 押上Skytree
  4. 秋叶原JR/步行
  5. 日本桥晚餐
蛇の市 18:30，建议 18:10 前到店附近。
西线

原宿/涩谷 → 池尻大桥

山手线逛西侧，涩谷收尾后坐东急田园都市线去六甲園。

  1. 五反田山手线
  2. 原宿步行
  3. 表参道步行/Metro
  4. 涩谷轻逛
  5. 池尻大桥烧肉
20:00 晚餐；19:15-19:25 从涩谷出发比较稳。

Daily Plan

每日安排

6/18 周四

到达日：五反田 + 涩谷轻松逛

  1. 羽田京急
  2. 品川换乘
  3. 五反田入住
  4. 涩谷轻逛
  1. 15:50抵达羽田机场。
  2. 17:30-18:30到五反田酒店入住/放行李，顺手确认 Suica/PASMO 和移动网络。
  3. 19:00-21:00涩谷 PARCO、Pokémon Center SHIBUYA、Nintendo Tokyo，按体力决定停留。
交通羽田到五反田约 35-45 分钟；涩谷往返走 JR 山手线最直接。
吃饭不放硬预约，五反田/涩谷随缘吃，给入境和行李留缓冲。
检查当天把 Disney App、门票二维码、移动支付和交通卡都确认一遍。

第一晚不放硬预约，晚餐随缘吃五反田或涩谷。

6/19 周五

Tokyo Disneyland：美女与野兽主日

  1. 五反田山手线
  2. 东京京叶线
  3. 舞浜Disney
  4. 五反田返程
  1. 07:15-07:30从五反田出发，东京站换京叶线要走一段，别把换乘估太短。
  2. 09:00-21:00Disneyland 全天；门票已订，开园时间以当天官网/App 为准。
  3. 入园后第一时间在 App 买 Beauty and the Beast Disney Premier Access。
  4. 20:30 前后看当天烟花/夜间演出安排，若太累可以提前一班回城。
核心目标美女与野兽 DPA 入园后立刻买；其他项目围绕它排。
交通舞浜返程人多，回五反田保守按 60 分钟估算。
当天准备移动电源、防晒/雨具、护照照片页备份；园内吃饭不额外预约。

园内解决晚餐，回五反田后轻松收尾。

6/20 周六

东京东线：浅草 + 晴空塔 + 秋叶原 + 蛇の市

  1. 五反田浅草线
  2. 浅草寺社
  3. 押上Skytree
  4. 秋叶原卡牌
  5. 日本桥寿司
  1. 09:00-09:35五反田出发，优先都营浅草线直达浅草。
  2. 09:35-11:15浅草寺、仲见世通、隅田川；人多时只保留浅草寺和河岸。
  3. 11:30-13:30Solamachi + Pokémon Center SKYTREE TOWN；展望台按天气临场决定。
  4. 14:00-17:15秋叶原卡牌店，重点 Hareruya2 Akihabara Tower，给付款/退税留时间。
  5. 17:35-18:10往日本桥/三越前方向移动，到店附近缓一口气。
  6. 18:30蛇の市本店晚餐，2 人，席のみ。
晴空塔只逛 Solamachi/Pokémon Center 不需要订票；天气好再买展望台。
雨天浅草缩短，Solamachi 和秋叶原室内时间加长。
硬约束18:30 寿司，建议 18:10 前到店附近，不把秋叶原拖太晚。

晴空塔展望台可按天气决定，不是必买项。

6/21 周日

东京西线：原宿/表参道 + 涩谷 + 六甲園

  1. 五反田山手线
  2. 原宿明治神宫
  3. 表参道步行
  4. 涩谷日落
  5. 池尻大桥烧肉
  1. 10:00-11:30明治神宫；如果下雨就压缩为参道散步，不硬逛。
  2. 11:30-15:30原宿、表参道、GYRE/Omotesando Hills 一带，午餐机动。
  3. 15:30-17:30涩谷 PARCO、Loft、Tower Records，提前整理购物袋。
  4. 17:30-19:10下午茶、咖啡或日落散步机动安排，不再排固定观景台。
  5. 19:15-19:40涩谷坐东急田园都市线到池尻大桥，再步行去六甲園；若从六本木/麻布台出发则优先打车。
  6. 20:00焼肉 六甲園晚餐，2 人；迟到风险比普通晚餐高。
节奏下午不再卡观景台，优先把表参道/涩谷逛舒服。
交通涩谷到池尻大桥很近，但晚高峰和步行要留 25-35 分钟。
备选如果临时订高空 lounge，看完日落后 19:10 左右就要撤。

20:00 六甲園是当天唯一硬节点，下午安排保持机动。

6/22 周一

返程日：不排景点

  1. 五反田退房
  2. 品川换乘
  3. 羽田机场
  1. 08:30-09:30早餐、收拾行李。
  2. 10:00从五反田出发去羽田，常规走品川换京急。
  3. 11:00 前后到羽田，办理值机/托运，留足购物和安检时间。
  4. 13:30航班起飞。
原则返程日不塞景点，避免拖着行李横跳。
交通五反田到羽田保守按 45-60 分钟估算。
检查前一晚确认护照、退税品、充电器和航班航站楼。

返程日保持简单，避免最后一刻赶行程。
